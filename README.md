# Deep Learning Real-Time ANPR Project
### Developed by: Harsh Jayeshbhai Patel

This is a personal precision Automatic Number Plate Recognition (ANPR) system using YOLOv8 and Sort Tracker with persistent memory and high-accuracy visual enhancements.

## Features
- **Real-time Visualization**: High-quality display with green corner boxes and white plate crops.
- **Persistent Tracking**: Plate numbers stay with the vehicle throughout the video.
- **Accuracy Boost**: High-confidence memory system that prevents OCR errors.

## Running the Project
```bash
python main.py
```
